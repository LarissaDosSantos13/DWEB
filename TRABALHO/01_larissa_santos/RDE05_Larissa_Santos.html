HIPERLINKS

MARCAÇÃO DE HIPERLINKS

Esta marcação e feita através de uma tag para referenciar o conteúdo desejado

HIPERLINKS EXTERNOS E REMOTOS

Para refencias sites e páginas da web utilizado o URL para referenciar o que desejarmos

HIPERLINKS INTERNO

Aponta em lugares diferentes dentro da mesma página e possibilita acessar parágrafos ou seleção. No texto do próprio Html. Muito útil para os alunos pesquisarem com mais objetividade.

HIPERLINKS IMAGENS

É possível utilizar um Hiperlink também com imagens com a utilização da tag.

HIPERLINKS LOCAIS

Os que apontam para outras páginas dentro de um mesmo site

PARA DOWNLOAD DE ARQUIVOS

São links que apontam para fazer o download de determinado arquivo.
